# Tiny Loot Quest

A simple top-down 2D game built with vanilla JavaScript and HTML5 Canvas.

## Requirements

A modern web browser with JavaScript enabled.

## How to play

Open `index.html` in your browser.

### Controls

- **WASD** or **Arrow Keys** - Move the player
- **SPACE** - Restart (when game over)

### Objective

- Collect gold coins to increase your score
- Avoid red enemies coming from all sides
- Pick up green power-ups for a temporary shield
- The game gets harder as your score increases
